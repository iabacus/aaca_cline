## 성능 비교

### 속도 vs 품질 트레이드오프

| 우선순위 | 추천 모델 | 이유 |
|----------|------------------|-----|
| **속도** | Cerebras의 Qwen3 Coder | 가장 빠른 추론 |
| **품질** | Claude Sonnet 4.5 | 복잡한 작업에 가장 신뢰도 높음 |
| **균형** | DeepSeek V3 | 저비용에서 좋은 품질 |

### 도구 신뢰성

도구 사용 신뢰성 순위:
1. **Claude Sonnet 4.5** - 가장 안정적인 도구 실행
2. **GPT-5** - 매우 우수하지만 가끔 포맷 이슈
3. **Gemini 2.5 Pro** - 표준 도구에 강함
4. **DeepSeek V3** - 기본 도구에 신뢰 가능
5. **Qwen3 변형** - 복잡한 도구는 재시도 필요할 수 있음

## 프로바이더 비교

### 프로바이더 기능

| 프로바이더 | 사용 가능한 모델 | 결제 | API 안정성 | 지원 |
|----------|-----------------|---------|---------------|---------|
| **Cline** | Multiple | 크레딧 기반 | 높음 | 인앱 |
| **Anthropic** | Claude 전용 | 사용량 기반 | 높음 | 이메일 |
| **OpenRouter** | 100+ 모델 | 사용량 기반 | 높음 | Discord |
| **OpenAI** | GPT 전용 | 사용량 기반 | 높음 | Forum |
| **로컬(Ollama)** | 오픈 소스 | 무료 | 해당 없음 | 커뮤니티 |

### 프로바이더 선택 가이드

다음 기준으로 프로바이더를 선택하세요:
- **간편함**: Cline(API 키 관리 없음)
- **다양성**: OpenRouter(모든 모델 접근)
- **직접 접근**: 개별 프로바이더(Anthropic, OpenAI)
- **프라이버시**: Ollama 또는 LM Studio(로컬 모델)
