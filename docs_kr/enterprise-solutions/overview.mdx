Cline Enterprise는 수백만 개발자가 사용하는 동일한 오픈 소스 아키텍처에 중앙 거버넌스를 제공합니다. 코드는 여러분의 환경에 그대로 남고, 협상된 요금으로 자체 추론을 사용하며, 플랫폼 팀이 조직 전체 배포에 필요한 보안과 관측성 기능을 제공합니다.

## 제공되는 핵심 가치

플랫폼 팀이 프로덕션 배포에 필요로 하는 5가지 핵심 기능을 제공합니다. 각 기능은 조직에서 AI 코딩을 확장하는 데 필요한 요건을 해결합니다.

### 설계 단계부터 보안

코드는 여러분의 환경 밖으로 나가지 않습니다. Cline은 모든 처리를 로컬에서 수행합니다. 업로드도, 인덱싱도, 데이터 학습도 없습니다.

### Bring Your Own Inference

기존 클라우드 계약과 협상된 요금을 그대로 사용하세요. 대부분의 AI 도구는 마진이 붙은 자체 추론을 강제합니다. Cline은 제공자에 직접 연결합니다.

지원 제공자:
- AWS Bedrock
- Google Vertex AI
- Azure OpenAI
- Anthropic direct
- OpenAI direct
- Cerebras
- 모든 OpenAI 호환 엔드포인트

새 모델이 나오면 즉시 전환할 수 있습니다. 일상 작업은 Claude Sonnet 4.5, 복잡한 리팩터링은 GPT-5, 단순 작업은 오픈 소스 모델을 사용하세요. 기존 클라우드 크레딧과 스타트업 프로그램 계약을 그대로 AI 코딩에 활용할 수 있습니다. 에이전트 루프는 Cline이 담당하고, 추론은 여러분이 관리합니다. 마진 없음, 벤더 락인 없음.

### 대규모 거버넌스

수천 명의 개발자가 AI를 사용할 때 플랫폼 팀은 중앙 제어가 필요합니다. 개인 API 키가 노트북에 흩어져 있으면 보안 리스크와 비용 초과가 발생합니다.

엔터프라이즈 거버넌스 제공 기능:
- **SSO 인증**: 개인 API 키 대신 회사 계정 사용
- **역할 기반 접근 제어**: 3단계 계층(Member/Admin/Owner)과 조직 범위 권한
- **모델 및 도구 제어**: 팀별 접근 가능한 모델과 도구 관리
- **원격 구성**: 하나의 대시보드에서 모든 개발자 설정 관리
- **사용량 추적 및 관측성**: OpenTelemetry 통합으로 사용량, 비용, 성능 모니터링 및 관리자 작업에 대한 선택적 감사 로깅

한 번 설정하면 어디든 배포됩니다. 개발자는 원하는 방식으로 작업하고, 여러분은 제어권을 유지합니다.

### 완전한 관측성

기존 관측성 스택으로 로그를 내보내고, 모든 팀의 사용량/비용/성능을 추적하세요.

- **OpenTelemetry 내보내기**: Datadog, Grafana, Splunk와 직접 통합
- **실시간 분석**: 도입률, 성능, 패턴 추적
- **비용 분해**: 팀별/모델별 비용을 정확히 파악
- **JSON 출력**: 기존 도구로 커스텀 대시보드 구축

프로덕션 시스템에 요구되는 동일한 관측성 기준을 제공합니다.

## 배포

Cline Enterprise는 인프라에 안전하게 연결됩니다. 클라우드 환경에 배포하고 기존 보안 정책과 컴플라이언스 요구사항에 맞게 구성할 수 있습니다.

조직 배포 절차:
1. Cline Core가 인프라에 연결되도록 구성
2. SSO, RBAC, 거버넌스 정책 설정
3. 기존 소프트웨어 배포 방식으로 개발자에게 배포
4. 관측성 도구로 사용량 모니터링
