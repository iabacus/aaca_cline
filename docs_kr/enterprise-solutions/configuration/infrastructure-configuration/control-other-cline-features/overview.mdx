이 섹션에서는 엔터프라이즈 배포에서 Cline의 동작을 제어하기 위한 구성 옵션을 다룹니다.

## 사용 가능한 설정

<Card title="YOLO 모드" icon="rocket" href="/enterprise-solutions/configuration/infrastructure-configuration/control-other-cline-features/yolo-mode">
  완전 자동 승인(자율 실행) 모드에 대한 엔터프라이즈 접근을 제어합니다.
</Card>

## 구성 방법

다음 방법으로 설정을 구성할 수 있습니다:

### 개인 사용자
- 사용자는 로컬 Cline 인터페이스에서 설정을 토글할 수 있습니다.
- 엔터프라이즈 정책으로 특정 설정을 제한할 수 있습니다.
- 변경 사항은 새 작업에 즉시 적용됩니다.

## 엔터프라이즈 제어

관리자는 원격 구성을 통해 정책을 강제할 수 있습니다:

```json
{
  "yoloModeAllowed": false
}
```

`yoloModeAllowed`를 `false`로 설정하면 사용자는 로컬 Cline 인터페이스에서 YOLO 모드를 활성화할 수 없습니다.
