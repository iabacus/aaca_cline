## Cline 텔레메트리

Cline은 익명 사용량 추적을 위한 옵트인 텔레메트리를 제공합니다:

- 기능 사용 패턴
- 작업 완료율  
- 오류 발생
- 성능 지표

사용자는 Cline 설정에서 텔레메트리를 켜거나 끌 수 있습니다. 모든 데이터는 익명이며 코드 내용, 파일 경로, 민감 정보는 포함하지 않습니다.

자세한 구성은 [Cline 텔레메트리](/enterprise-solutions/monitoring/telemetry)를 참고하세요.

## OpenTelemetry 통합

고급 모니터링이 필요하다면 Cline은 OTLP(OpenTelemetry Protocol)로 메트릭과 로그를 자체 인프라로 내보낼 수 있습니다.

이를 통해:
- 기존 관측성 플랫폼으로 텔레메트리 내보내기
- Datadog, New Relic, Grafana Cloud 등과 통합
- 모니터링 데이터에 대한 완전한 제어
- 조직 전반의 메트릭 집계

<Note>
OpenTelemetry 통합은 **선택 사항**이며 추가 구성이 필요합니다. 대부분의 사용자는 이 기능이 필요하지 않습니다.
</Note>

설정 방법은 [OpenTelemetry](/enterprise-solutions/monitoring/opentelemetry)를 참고하세요.

## 사용 사례

### Cline 텔레메트리를 사용할 때
- 익명 사용량 데이터를 통해 Cline 개선에 기여하고 싶을 때
- 추가 설정 없이 사용하고 싶을 때
- 대부분의 사용자에게 적합

### OpenTelemetry를 사용할 때
- 자체 시스템에서 세밀한 메트릭이 필요할 때
- 기존 관측성 인프라와 통합해야 할 때
- 디버깅을 위한 상세 로그/메트릭이 필요할 때
- 커스텀 대시보드 또는 알림이 필요할 때
