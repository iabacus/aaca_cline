## Cline은 무엇을 할 수 있나요?

Cline은 다음을 수행할 수 있는 AI 어시스턴트입니다:

- 프로젝트에서 파일 편집 및 생성
- 터미널 명령 실행
- 코드 검색 및 분석
- 디버깅 및 문제 해결 지원
- 반복 작업 자동화
- 외부 도구와 통합


## 사용 가능한 도구

최신 구현 상세는 [Cline 저장소](https://github.com/cline/cline/blob/main/src/core/prompts/system-prompt/tools)의 전체 소스 코드를 참고하세요.

Cline은 다양한 작업을 위해 다음 도구에 접근할 수 있습니다:

1. **파일 작업**

    - `write_to_file`: 파일 생성 또는 덮어쓰기
    - `read_file`: 파일 내용 읽기
    - `replace_in_file`: 파일 내 특정 부분 편집
    - `search_files`: 정규식으로 파일 검색
    - `list_files`: 디렉터리 내용 나열

2. **터미널 작업**

    - `execute_command`: CLI 명령 실행
    - `list_code_definition_names`: 코드 정의 목록 출력

3. **MCP 도구**

    - `use_mcp_tool`: MCP 서버의 도구 사용
    - `access_mcp_resource`: MCP 서버 리소스 접근
    - 사용자가 커스텀 MCP 도구를 만들어 Cline이 접근하도록 할 수 있음
    - 예: 날씨 API 도구를 만들어 Cline이 예보를 조회하도록 사용

4. **인터랙션 도구**
    - `ask_followup_question`: 사용자에게 추가 질문
    - `attempt_completion`: 최종 결과 제시


각 도구는 특정 파라미터와 사용 패턴을 가집니다. 예시:

-   새 파일 생성(`write_to_file`):

    ```xml
    <write_to_file>
    <path>src/components/Header.tsx</path>
    <content>
    // Header component code
    </content>
    </write_to_file>
    ```

-   패턴 검색(`search_files`):

    ```xml
    <search_files>
    <path>src</path>
    <regex>function\s+\w+\(</regex>
    <file_pattern>*.ts</file_pattern>
    </search_files>
    ```

-   명령 실행(`execute_command`):
    ```xml
    <execute_command>
    <command>npm install axios</command>
    <requires_approval>false</requires_approval>
    </execute_command>
    ```

## 공통 작업

1. **새 컴포넌트 생성**

    - "src/utils/format.ts 오류를 수정해줘"

3. **코드 리팩터링**

    - "Button 컴포넌트를 TypeScript로 리팩터링해줘"

4. **명령 실행**
    - "npm install로 axios 추가해줘"
