## Cline에서 LM Studio 설정하기

LM Studio로 로컬에서 AI 모델을 실행해 Cline과 함께 사용합니다.

### 사전 준비

-   AVX2를 지원하는 Windows, macOS, Linux 컴퓨터
-   VS Code에 Cline 설치

### 설정 단계

#### 1. LM Studio 설치

-   [lmstudio.ai](https://lmstudio.ai)에 접속
-   운영체제에 맞는 버전 다운로드 및 설치

#### 2. LM Studio 실행

-   설치된 앱을 실행합니다
-   왼쪽에 네 개의 탭이 보입니다: **Chat**, **Developer**(서버 시작), **My Models**(다운로드한 모델 저장), **Discover**(새 모델 추가)

### 추천 모델 및 설정

Cline에 가장 적합한 모델은 **Qwen3 Coder 30B A3B Instruct**입니다. 강력한 코딩 성능과 안정적인 도구 사용을 제공합니다.

#### 핵심 설정

Developer 탭에서 모델을 로드한 뒤 다음 설정을 구성합니다:

1. **Context Length**: 262,144로 설정(모델 최대치)
2. **KV Cache Quantization**: 체크 해제(일관된 성능에 중요)
3. **Flash Attention**: 가능하면 활성화(성능 향상)

#### 양자화 가이드

RAM 용량에 따라 양자화를 선택하세요:

- **32GB RAM**: 4-bit 양자화(~17GB 다운로드)
- **64GB RAM**: 8-bit 양자화(~32GB 다운로드)로 더 높은 품질
- **128GB+ RAM**: 풀 프리시전 또는 더 큰 모델 고려

#### 모델 형식

- **Mac(Apple Silicon)**: 성능 최적화를 위해 MLX 형식 사용
- **Windows/Linux**: GGUF 형식 사용

### Compact Prompts 활성화

로컬 모델에서 최적 성능을 위해 Cline 설정에서 compact prompts를 활성화하세요. 핵심 기능을 유지하면서 프롬프트 크기를 90% 줄입니다.

Cline Settings → Features → Use Compact Prompt로 이동해 토글을 켭니다.

### 중요 참고사항

-   Cline과 함께 사용하기 전에 LM Studio를 시작하세요
-   LM Studio를 백그라운드에서 실행 상태로 유지하세요
-   첫 모델 다운로드는 크기에 따라 수 분이 걸릴 수 있습니다
-   모델은 다운로드 후 로컬에 저장됩니다

### 문제 해결

1. Cline이 LM Studio에 연결하지 못하면:
2. LM Studio 서버가 실행 중인지 확인(Developer 탭)
3. 모델이 로드되었는지 확인
4. 시스템이 하드웨어 요구사항을 충족하는지 확인
