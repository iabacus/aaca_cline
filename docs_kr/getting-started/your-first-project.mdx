Cline의 실제 동작을 보고 싶나요? 이 실습 튜토리얼은 1분 안에 웹사이트를 만드는 과정을 안내합니다. Cline이 요구사항을 이해하고, 파일을 생성하고, 피드백에 따라 반복하는 과정을 자연스러운 대화로 경험할 수 있습니다.

이 가이드를 마치면 작동하는 웹사이트를 만들고 Cline과 작업하는 기본기를 익히게 됩니다.

## 사전 준비

- 에디터에 **Cline 설치**([설치 가이드](/getting-started/installing-cline))
- **AI 모델 선택**([모델 설정](/getting-started/selecting-your-model))
- 에디터에서 **아무 폴더나 열기**(또는 새 빈 폴더 생성)

## 1단계: Cline 열기

에디터 왼쪽 사이드바의 Cline 아이콘을 클릭합니다. 채팅 패널이 열립니다.

<Info>
**빠른 팁:** `Cmd+Shift+P`(Mac) 또는 `Ctrl+Shift+P`(Windows/Linux)로 Command Palette를 열고 "Cline: Open In New Tab"을 검색해도 됩니다.
</Info>

## 2단계: Cline에게 작업 요청

다음 프롬프트를 Cline 채팅에 복사해 붙여넣으세요:

```
단일 HTML 파일로 간단한 웹사이트를 만들어줘. 다음을 포함해야 해:
- "Hello from Cline!"라고 표시되는 환영 메시지
- 컬러풀한 그라디언트 배경
- 클릭할 때마다 다른 색상 테마로 바뀌는 버튼
- 현대적이고 깔끔한 디자인
- 모든 CSS와 JavaScript는 동일한 HTML 파일에 포함
```

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/chat-prompt.png" alt="Cline 채팅 프롬프트"/>
</Frame>

Enter를 누르고 Cline이 작업하는 것을 확인하세요!

## 3단계: 다음에 일어나는 일

Cline은 다음을 수행합니다:

1. **단일 파일 생성**:
   - `index.html` - CSS와 JavaScript가 내장된 완전한 웹페이지

2. **승인 요청**(자동 승인 켜짐 제외)
   - "Approve"를 클릭해 Cline이 파일을 만들도록 허용
   - 실행 전 계획을 검토할 수 있음

3. **작업 완료**(몇 초 내)

## 4단계: 웹사이트 보기

Cline이 완료하면:

1. 에디터 파일 탐색기에서 **`index.html` 찾기**
2. **우클릭** 후 선택:
   - "Reveal in Finder/Explorer" → 브라우저에서 더블클릭으로 열기
3. **버튼 클릭**해 색상 테마가 바뀌는지 확인

## 변경 요청해 보기

같은 채팅에서 다음을 요청해 보세요:

```
버튼이 눌린 횟수를 보여주는 카운터를 추가해줘
```

또는

```
페이지가 로드될 때 환영 메시지가 페이드인 되게 해줘
```

Cline은 이전 대화 컨텍스트를 이해하고 파일을 업데이트합니다.

<Tip>
**이제 다음을 할 수 있습니다:**
- 명확한 프롬프트로 Cline에게 작업 지시
- Cline의 작업을 검토하고 승인
- 몇 초 만에 완전한 프로젝트 구축
- 기존 작업을 반복 개선
</Tip>
