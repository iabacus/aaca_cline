Cline은 요청을 이해하고 코드를 작성하기 위해 AI 모델이 필요합니다. 어떤 전문가와 협업할지 고르는 것과 같습니다. 모델마다 강점과 비용이 다릅니다.

## 빠른 시작: 프로바이더 선택

가장 쉬운 시작 방법은 **Cline**을 프로바이더로 선택하는 것입니다:

1. **Cline 설정 열기**: Cline 채팅 오른쪽 상단의 톱니바퀴(⚙️) 클릭
2. API Provider 드롭다운에서 **"Cline"** 선택
3. 드롭다운에서 **모델 선택** — **Claude Sonnet 4.5** 또는 **DeepSeek V3**로 시작을 권장합니다

<Frame>
  <img src="https://storage.googleapis.com/cline_public_images/step2-provider.png" alt="Cline 프로바이더 선택" />
</Frame>

**끝!** API 키 관리 없이 여러 모델에 접근할 수 있습니다.

<Tip>
**무료 모델 제공**: Cline은 파트너 프로바이더를 통해 간헐적으로 무료 추론을 제공합니다. 제공 시 모델 드롭다운에서 옵션이 표시됩니다.
</Tip>

## 대안: 다른 프로바이더 사용

자신의 계정이나 API 키를 사용하고 싶다면 다음 옵션이 있습니다:

### OpenAI로 로그인(권장)

OpenAI 모델을 사용하는 가장 쉬운 방법은 **OpenAI Codex**입니다—API 키가 필요 없습니다:

1. API Provider 드롭다운에서 **"OpenAI Codex"** 선택
2. **"Sign in with OpenAI"** 클릭
3. 브라우저에서 Cline 권한 승인
4. 모델 선택

<Card title="OpenAI Codex 설정 가이드" icon="key" href="/provider-config/openai-codex">
  스크린샷과 문제 해결 팁이 포함된 전체 설정 가이드를 확인하세요.
</Card>

### 다른 프로바이더

다음 프로바이더는 API 키를 통해 사용할 수 있습니다:

- **OpenRouter** - 가성비 좋고 모델 다양
- **Anthropic** - Claude 모델 직접 접근  
- **OpenAI** - API 키로 GPT 모델 사용
- **Google Gemini** - Google AI 모델
- **Ollama** - 로컬에서 모델 실행

프로바이더를 선택한 후에는:
1. 해당 사이트에서 API 키 발급
2. Cline 설정의 API Key 필드에 입력
3. 모델 선택

<Note>
대부분의 프로바이더는 API 키 발급 전에 결제 정보 등록이 필요합니다.
</Note>

## 어떤 모델을 선택해야 할까?

처음 시작한다면 다음을 추천합니다:

| 우선순위 | 추천 모델 | 이유 |
|---------------|-------------------|-----|
| **신뢰성** | Claude Sonnet 4.5 | 코딩 작업에서 가장 안정적 |
| **가성비** | DeepSeek V3 | 낮은 비용 대비 높은 성능 |
| **속도** | Qwen3 Coder | 빠른 응답 |
| **프라이버시** | Ollama 모델 | 로컬 실행 |

모델은 언제든 바꿀 수 있으며 대화는 유지됩니다.
