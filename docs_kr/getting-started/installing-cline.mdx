## 시작 전 준비

<CardGroup cols={1}>
  <Card title="계정 만들기" icon="user-plus" href="https://app.cline.bot/login">
    **무료 Cline 계정**을 만들면 다음을 사용할 수 있습니다:
    - stealth 모델을 포함한 다양한 AI 모델 접근
    - API 키 관리 없이 매끄러운 설정
    - 파트너 프로바이더를 통한 간헐적 무료 추론
  </Card>
  
  <Card title="호환 에디터 준비" icon="code">
    Cline은 다음 에디터에서 동작합니다:
    - **VS Code** / **Cursor**
    - **JetBrains IDEs** (IntelliJ, PyCharm, WebStorm 등)
    - **VSCodium** / **Windsurf**
    
    진행 전에 하나를 설치하세요.
  </Card>
</CardGroup>

<Tabs>
  <Tab title="VS Code/Cursor" icon="code">
    <Frame>
      <img
      	src="https://storage.googleapis.com/cline_public_images/vscode.png"

    <Steps>
      <Step title="확장 열기">
        VS Code를 실행하고 Extensions 뷰를 엽니다:
        - `Ctrl/Cmd + Shift + X` 또는
        - Activity Bar의 Extensions 아이콘 클릭
      </Step>

      <Step title="Cline 검색">
        Extensions 마켓플레이스 검색창에 **"Cline"** 입력
        	<Frame>
            <img src="https://storage.googleapis.com/cline_public_images/extension-installation.png" alt="VS Code 마켓플레이스에서 Cline 확장 검색" 
            />
          </Frame>
      </Step>

      <Step title="설치">
        Cline 확장에서 **Install** 버튼 클릭
        
        <Frame caption="VS Code 마켓플레이스에서 Cline 확장 설치 준비 화면">
        	<img
        		src="https://storage.googleapis.com/cline_public_images/docs/assets/image%20(20).png"
        		alt="VS Code 마켓플레이스에서 Cline 확장 표시"
        	/>
        </Frame>
      </Step>

      <Step title="Cline 열기">
        설치가 완료되면:
        - Activity Bar에서 **Cline 아이콘** 클릭 또는
        - Command Palette 열기(`Ctrl/Cmd + Shift + P`) → **"Cline: Open In New Tab"** 입력

        <Frame caption="VS Code에서 열린 Cline">
        	<img
        		src="https://storage.googleapis.com/cline_public_images/after-installation.png"
        		alt="VS Code에서 열린 Cline"
        	/>
        </Frame>
        
        <Warning>
        VS Code에 "Running extensions might..." 대화상자가 표시되면 **Allow**를 클릭하세요. Cline 아이콘이 보이지 않으면 VS Code를 재시작하세요.
        </Warning>
      </Step>
    </Steps>

    <Check>
    **설치 완료!** 이제 에디터에서 Cline 인터페이스가 보여야 합니다. 로그인할 시간입니다!
    </Check>

    <Accordion title="문제 해결">

    **플러그인 설치 문제**

    마켓플레이스에서 Cline이 보이지 않는다면:
    - **Marketplace** 탭(Installed가 아님)에서 검색 중인지 확인
    - "Cline AI" 또는 "Cline"으로 다시 검색
    - VS Code 버전 호환 여부 확인

    설치 실패 시:
    - VS Code를 재시작하고 다시 시도
    - 인터넷 연결 확인
    - 대안으로 VSIX 파일 설치 시도

    **플러그인이 보이지 않음**

    설치 후 Cline 툴 창이 보이지 않는다면:
    - VS Code를 완전히 재시작(File → Exit 후 재실행)
    - **View** → **Command Palette** → "Cline: Open In New Tab" 확인
    - **Extensions** 뷰에서 플러그인이 활성화되어 있는지 확인
    - Activity Bar에서 Cline 아이콘 확인(보통 왼쪽)

    **일반적인 문제**

    플러그인은 설치된 것 같지만 작동하지 않는다면:
    - 설치 후 VS Code 재시작 여부 확인
    - Developer Console에 오류 메시지가 있는지 확인
    - 확장을 비활성화했다가 다시 활성화

    성능 또는 호환성 문제:
    - 지원되는 VS Code 버전을 사용 중인지 확인
    - 호환성을 개선할 수 있는 VS Code 업데이트 확인
    - 필요 시 리소스를 많이 사용하는 다른 확장 닫기

    도움이 필요하신가요? [Discord 커뮤니티](https://discord.gg/cline)에 참여하세요.

    </Accordion>

    ### 설치 방법 선택

    <Tabs>
      <Tab title="IDE에서 설치(권장)">
        <Steps>
          <Step title="설정 열기">
            JetBrains IDE에서 **Settings**로 이동:
            - Windows/Linux: `Ctrl+Alt+S`
            - macOS: `Cmd+,`
          </Step>

          <Step title="플러그인으로 이동">
            **Plugins** → **Marketplace** 탭으로 이동
          </Step>

          <Step title="Cline 설치">
            **"Cline"**을 검색하고 **Install** 클릭
            
            <Frame>
            	<img
            		src="https://storage.googleapis.com/cline_public_images/jetbrains-installation.png"
            		alt="JetBrains 마켓플레이스에서 Cline 플러그인 검색 결과"
            	/>
            </Frame>
          </Step>

          <Step title="IDE 재시작">
            설치 완료를 위해 IDE를 재시작합니다
            <Frame>
            	<img
            		src="https://storage.googleapis.com/cline_public_images/jetbrains-post-installation.png"
            		alt="JetBrains 마켓플레이스에서 Cline 플러그인 검색 결과"
            	/>
            </Frame>
          </Step>
        </Steps>
      </Tab>

      <Tab title="브라우저 설치">
        <Steps>
          <Step title="마켓플레이스 방문">
            [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/28247-cline)로 이동
          </Step>

          <Step title="Install to IDE 클릭">
            **Install to IDE** 버튼을 클릭
          </Step>

          <Step title="IDE에서 확인">
            IDE가 열리며 설치 확인을 요청합니다
          </Step>

          <Step title="IDE 재시작">
            설치 완료를 위해 재시작
          </Step>
        </Steps>
      </Tab>

      <Tab title="수동 설치">
        <Steps>
          <Step title="플러그인 다운로드">
            [마켓플레이스 페이지](https://plugins.jetbrains.com/plugin/28247-cline)에서 다운로드
          </Step>

          <Step title="설정 열기">
            **Settings** → **Plugins**로 이동
          </Step>

          <Step title="디스크에서 설치">
            톱니바퀴 아이콘 → **Install Plugin from Disk** 클릭
          </Step>

          <Step title="파일 선택 및 재시작">
            다운로드한 `.zip` 파일을 선택하고 IDE를 재시작
          </Step>
        </Steps>
      </Tab>
    </Tabs>

    <Check>
    **설치 완료!** **View** → **Tool Windows** → **Cline**에서 Cline을 찾으세요(보통 오른쪽).
    </Check>

    ### JetBrains에서 동작하는 기능

    <AccordionGroup>
      <Accordion title="✅ 모든 핵심 기능" defaultOpen>
        - Diff 편집 및 파일 수정
        - 다중 API 프로바이더(Anthropic, OpenAI, 로컬 모델)
        - MCP 서버 및 커스텀 도구
        - Cline rules 및 workflows
        - 파일/폴더/문제 @ 멘션
        - 드래그 앤 드롭 지원
      </Accordion>

      <Accordion title="📋 터미널 통합 차이">
        **JetBrains는 VS Code와 다르게 터미널 출력을 표시합니다:**
        - VS Code: 출력이 채팅으로 바로 스트리밍
        - JetBrains: 출력이 접을 수 있는 "Command Output" 섹션에 표시
        
        두 환경 모두 명령 실행은 성공하며, JetBrains에서는 섹션을 펼치면 결과가 보입니다.
      </Accordion>
    </AccordionGroup>

    <Accordion title="문제 해결">

    **플러그인 설치 문제**

    마켓플레이스에서 Cline이 보이지 않는다면:
    - **Marketplace** 탭(Installed가 아님)에서 검색 중인지 확인
    - "Cline AI" 또는 "Cline"으로 다시 검색
    - IDE 버전이 호환되는지 확인(2023.1 이상 권장)

    설치 실패 시:
    - IDE를 재시작하고 다시 시도
    - 인터넷 연결 확인
    - 대안으로 디스크에서 설치 시도

    - IDE를 완전히 재시작(File → Exit 후 재실행)
    - **View** → **Tool Windows** → **Cline** 확인
    - **Settings** → **Plugins** → **Installed** 탭에서 플러그인 활성화 확인
    - IDE의 툴 윈도 바에서 Cline 아이콘 확인(보통 오른쪽)

    **일반적인 문제**

    플러그인은 설치된 것 같지만 작동하지 않는다면:
    - 설치 후 IDE 재시작 여부 확인
    - IDE 이벤트 로그에 오류가 있는지 확인
    - Settings에서 플러그인을 비활성화 후 재활성화

    성능 또는 호환성 문제:
    - 지원되는 JetBrains IDE 버전을 사용 중인지 확인
    - 호환성을 개선할 수 있는 IDE 업데이트 확인
    - 필요 시 IDE 메모리 할당을 늘리기

    도움이 필요하신가요? [Discord 커뮤니티](https://discord.gg/cline)에 참여하세요.

## 다음 단계: 로그인하고 시작하기

<Steps>
  <Step title="Cline 열기">
    에디터에서 Cline을 찾아 엽니다:
    - **VS Code/Cursor/VSCodium/Windsurf:** Activity Bar에서 Cline 아이콘 클릭
    - **JetBrains:** **View** → **Tool Windows** → **Cline**
  </Step>

  <Step title="로그인">
    Cline 인터페이스에서 **Sign Up** 버튼을 클릭
    
    <Info>
    [app.cline.bot](https://app.cline.bot)로 리디렉션되어 인증을 진행합니다. 로그인 후 자동으로 에디터로 돌아옵니다.
    </Info>
    <Info>
    Cline 인증 페이지로 이동해 계정으로 로그인합니다.
    </Info>
  </Step>

  <Step title="준비 완료!">
    <Check>
    **축하합니다!** 이제 Cline을 사용할 준비가 완료되었습니다!
    
    Cline이 프로젝트 제작을 도와줄 준비가 되었습니다.
    </Check>
  </Step>
</Steps>
