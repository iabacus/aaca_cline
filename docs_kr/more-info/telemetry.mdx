title: "텔레메트리"

### 개요

Cline을 모두에게 더 나은 도구로 만들기 위해, 오픈 소스 AI 코딩 에이전트를 어떻게 사용하는지 이해할 수 있는 사용 데이터를 수집합니다. 이 피드백 루프는 Cline의 기능과 사용자 경험 개선에 핵심적입니다.

데이터 수집과 분석에는 오픈 소스 분석 플랫폼인 PostHog를 사용합니다. 텔레메트리 구현은 완전히 투명합니다. 무엇을 추적하는지 정확히 확인하려면 [소스 코드](https://github.com/cline/cline/blob/main/src/services/posthog/telemetry/TelemetryService.ts)를 검토할 수 있습니다.

### 추적 정책

프라이버시는 최우선입니다. 기본적으로 수집 데이터는 모두 익명 처리됩니다. Cline 계정으로 로그인하면, 제품 개선과 문제 발생 시 더 나은 지원을 위해 텔레메트리 데이터가 계정과 연결됩니다. 여러분의 코드, 프롬프트, 대화 내용은 항상 비공개이며 절대 수집되지 않습니다.

### 수집 항목

다음과 같은 기본 사용 데이터를 수집합니다:

**작업 상호작용:** 작업 시작/종료, 대화 흐름(내용 제외)\
**모드 및 도구 사용:** Plan/Act 모드 전환, 사용 중인 도구\
**토큰 사용량:** 비용 추정을 위한 대화 길이 메트릭(토큰 내용은 수집하지 않음)\
**시스템 컨텍스트:** OS 유형 및 VS Code 환경 세부정보\
**UI 활동:** 네비게이션 패턴과 기능 사용

완전한 투명성을 위해 [텔레메트리 구현 코드](https://github.com/cline/cline/blob/main/src/services/posthog/telemetry/TelemetryService.ts)에서 실제 추적 이벤트를 확인할 수 있습니다.

### 옵트아웃 방법

Cline의 텔레메트리는 완전히 선택 사항입니다:

-   VS Code 확장을 설치하거나 업데이트할 때 텔레메트리 안내 메시지가 표시됩니다
-   설정에서 언제든지 선호를 변경할 수 있습니다

Cline은 VS Code의 전역 텔레메트리 설정도 존중합니다. VS Code에서 텔레메트리를 비활성화했다면 Cline의 텔레메트리도 자동으로 비활성화됩니다.
