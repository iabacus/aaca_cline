## 타협 없는 오픈 소스 AI 코딩

Cline은 제한 없이, 놀랄 일 없이, 특정 모델 생태계에 종속되지 않는 최전선 AI에 대한 직접적이고 투명한 접근을 제공합니다. 모든 결정을 확인하고, 어떤 모델이든 선택하며, 비용을 제어하세요.

<CardGroup cols={1}>
  <Card title="완전한 투명성" icon="eye">
    Cline이 파일을 읽고, 접근 방법을 검토하고, 변경을 제안하는 과정을 실시간으로 확인하세요. 모든 결정이 보이고, 모든 편집은 적용 전에 검토할 수 있습니다.
  </Card>

  <Card title="추론 선택" icon="sliders">
    복잡한 추론은 Claude, 대규모 컨텍스트는 Gemini, 효율은 Qwen3 Coder. 새 모델이 나오면 즉시 전환하세요. API 키와 선택권은 여러분에게 있습니다.
  </Card>

  <Card title="제로 트러스트 설계" icon="lock">

  </Card>
</CardGroup>

## Plan & Act 모드

Cline은 코드 작성 전에 코드베이스를 탐색하고, 여러분과 함께 종합적인 계획을 세워 프로젝트의 전체 컨텍스트를 이해하도록 합니다.

<Tabs>
  <Tab title="Plan 모드" icon="lightbulb">
    복잡한 작업에서 Cline이 코드베이스를 탐색하고, 명확화 질문을 하며, 변경 전에 상세 구현 계획을 작성합니다.

    - 정보 수집 및 컨텍스트 구축
    - 명확화 질문
    - 상세 실행 계획 작성
    - 접근 방식 논의
  </Tab>

  <Tab title="Act 모드" icon="play">
    계획을 승인하면, Cline이 완전한 투명성과 제어 아래에서 솔루션을 구현합니다.

    - 계획된 액션 실행
    - 도구로 파일 수정 및 명령 실행
    - 솔루션 구현
    - 결과 및 완료 피드백 제공
  </Tab>
</Tabs>
