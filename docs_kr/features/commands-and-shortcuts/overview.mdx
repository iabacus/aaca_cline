Cline은 VS Code 인터페이스에 직접 통합되어, 워크플로를 방해하지 않고 AI 도움을 받을 수 있습니다. 이러한 통합은 에디터 전반의 컨텍스트 메뉴, 키보드 단축키, 빠른 수정(Quick Fix)으로 나타납니다.

### 에디터 통합이란?

에디터 통합은 작업 중인 위치에서 바로 Cline을 사용할 수 있게 해주는 명령과 단축키입니다. 먼저 Cline 패널로 이동할 필요 없이 코드를 선택하고 우클릭해 즉시 도움을 받을 수 있습니다.
이 통합 기능은 VS Code 곳곳에 다음과 같이 나타납니다:

-   에디터 컨텍스트 메뉴(우클릭 메뉴) - "Add to Cline"
-   터미널 컨텍스트 메뉴 - "Add to Cline"
-   소스 컨트롤 뷰 - "Generate Commit Message"
-   키보드 단축키 - 다양한 Cline 명령
-   빠른 수정 옵션(전구 메뉴) - "Fix with Cline", "Explain with Cline", "Improve with Cline"

### 동작 방식

이 명령들을 사용하면 Cline은:

-   관련 컨텍스트(선택한 코드, 파일 경로, 터미널 출력 등)를 캡처
-   Cline 인터페이스에 포커스
-   캡처된 컨텍스트로 대화를 생성
-   경우에 따라 자동으로 제안 프롬프트를 생성

내부적으로는 VS Code 확장 API를 사용해 명령을 등록하고, 에디터 상태를 읽고, VS Code 인터페이스를 제어합니다.
