# Jupyter 노트북

Cline은 Jupyter 노트북(`.ipynb` 파일)을 폭넓게 지원하며, 셀 단위 컨텍스트를 완전히 이해하는 AI 보조 편집을 제공합니다. 이 기능은 Amazon과의 협업으로 개발되어 데이터 사이언스 워크플로에 AI 코딩 지원을 제공합니다.

## 시작하기

Jupyter 노트북 지원은 Cline의 내장 기능입니다. 사용하려면 VS Code에서 Jupyter 노트북 확장을 활성화하기만 하면 됩니다. 어떤 `.ipynb` 파일을 열면 노트북 인터페이스에 AI 보조 버튼이 나타납니다.

## 사용 방법

### 셀 생성

노트북 툴바의 반짝이 아이콘(✨)을 클릭하면 AI 보조로 새 셀을 생성할 수 있습니다.

![셀 생성 데모](../assets/jupyter-generate-cell.gif)

**동작 방식:** AI는 주변 셀의 컨텍스트를 받아 이미 범위에 들어온 변수와 import를 이해합니다. 따라서 기존 DataFrame, 함수, 기타 객체를 다시 설명하지 않고 참조할 수 있습니다.

**예시 프롬프트:** "수치 컬럼의 상관 행렬을 히트맵으로 시각화해줘"

셀은 올바른 노트북 JSON 구조로 삽입되며, 메타데이터를 유지하고 바로 실행할 수 있는 상태로 생성됩니다.

### 셀 설명

각 셀의 타이틀 바에서 Explain 버튼을 클릭하면 해당 셀의 동작을 자세히 설명해 줍니다.

다음에 유용합니다:

- 예전 노트북을 다시 볼 때
- 동료의 분석에 온보딩할 때
- 복잡한 변환을 이해할 때

**동작 방식:** Cline은 출력까지 포함한 전체 셀 컨텍스트를 추출하므로, 컬럼 이름, 행 수, 계산 결과 등 실제 결과를 기반으로 설명할 수 있습니다.

### 셀 개선

각 셀의 타이틀 바에서 Improve 버튼을 클릭하면 AI가 기존 셀을 개선해 줍니다.

![셀 설명 및 개선 데모](../assets/jupyter-explain-improve-cell.gif)

다음과 같은 작업에 활용하세요:

- 느린 pandas 연산 최적화
- 에러 처리 추가
- 가독성을 위한 리팩터링
- 루프를 벡터화 연산으로 변환

**동작 방식:** Cline은 노트북 구조에서 셀의 위치와 메타데이터를 유지하면서 개선안을 제시합니다. AI는 무엇이 바뀌었는지와 이유를 함께 설명합니다.

## 셀 컨텍스트 동작 방식

전통적인 파일 편집과 달리, Jupyter 노트북은 셀 배열로 구성된 JSON 문서입니다. 각 셀에는 유형, 소스 내용, 메타데이터, 실행 횟수, 출력이 포함됩니다.

Jupyter 명령을 사용할 때 Cline은 다음을 포함한 구조화된 컨텍스트를 추출합니다:

- **셀 유형**(code, markdown, raw)
- **소스 내용**(라인 배열)
- **셀 메타데이터** 및 구성
- **실행 횟수**(code 셀)
- **출력**(데이터, 텍스트, 에러 트레이스 포함)

이 구조화된 표현은 AI가 코드뿐 아니라 노트북 내 맥락과 실제 출력까지 이해하도록 해줍니다.

### JSON 구조 보존

Cline은 셀 JSON 구조를 신중하게 다루도록 설계되어 있으며, 다음을 목표로 합니다:

- 셀 경계를 유지
- 실행 횟수 보존
- 셀 메타데이터 유지
- 소스 셀과 출력의 연결 유지

AI는 노트북 구조를 보존하도록 명시적으로 지시되지만, 노트북 형식이 올바른지 항상 변경 사항을 검토하는 것을 권장합니다.

## 키보드 단축키

다음 명령을 키보드 단축키로 바인딩하면 더 빠르게 접근할 수 있습니다:

1. VS Code 키보드 단축키 열기(Cmd/Ctrl + K, Cmd/Ctrl + S)
2. `cline.jupyterGenerateCell`, `cline.jupyterExplainCell`, `cline.jupyterImproveCell` 검색
3. 원하는 키 조합 할당

## 더 좋은 결과를 위한 팁

**셀 생성:**
- 셀이 해야 할 일을 구체적으로 설명
- 기존 변수를 이름으로 참조(AI가 볼 수 있음)
- 선호 라이브러리가 있으면 명시(예: "seaborn 사용", "plotly 사용")

**셀 설명:**
- 실행된 셀에서 가장 잘 동작(출력이 추가 컨텍스트 제공)
- pandas groupby/merge처럼 체인형 연산을 이해하는 데 유용

**셀 개선:**
- 개선하고 싶은 측면을 명시(성능, 가독성, 에러 처리 등)
- AI가 제안한 변경 이유를 설명해 줌

## 제한 사항

- 노트북 지원은 VS Code에서 Jupyter 노트북 확장이 활성화되어 있어야 합니다
- 셀 컨텍스트 추출은 VS Code의 노트북 API에 의존합니다
- 매우 큰 노트북은 일부 모델에서 처리 가능한 컨텍스트를 초과할 수 있습니다
