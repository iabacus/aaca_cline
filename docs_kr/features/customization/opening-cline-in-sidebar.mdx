---

기본적으로 Cline을 처음 설치하면 VS Code의 왼쪽 사이드바에 파일 탐색기 및 다른 확장과 함께 표시됩니다. 하지만 더 나은 코딩 경험을 위해 Cline을 오른쪽 사이드바로 옮기는 것을 권장합니다. 이렇게 하면 왼쪽 사이드바에 프로젝트 파일을 계속 표시하면서, 오른쪽에서 Cline과 대화할 수 있어 Cline이 프로젝트를 작업하는 동안 코드베이스를 완전히 볼 수 있습니다.

## VS Code

오른쪽 사이드바에 Cline을 열려면:

<Steps>
  <Step title="확장 뷰 정렬">
    확장 뷰가 왼쪽에 세로로 정렬되어 있는지 확인하세요
  </Step>
  <Step title="오른쪽 뷰 열기">
    VS Code의 오른쪽 패널을 여는 버튼을 클릭하세요(보통 GitHub Copilot 채팅을 여는 버튼). 선택적으로 `Option + CMD/Ctrl + B` 단축키를 사용할 수 있습니다.
  </Step>
  <Step title="Cline 아이콘 드래그">
    Cline 아이콘을 오른쪽 뷰 상단의 네비게이션 패널로 드래그하세요
  </Step>
</Steps>

## Cursor

오른쪽 사이드바에 Cline을 열려면:

<Steps>
  <Step title="확장 정렬">
    Cursor는 기본적으로 AI 채팅 인터페이스 공간을 최적화하기 위해 가로 활동 표시줄을 사용합니다([자세한 내용](https://cursor.com/docs/configuration/migrations/vscode#activity-bar-orientation)). 세로로 전환하려면:
    
    1. 명령 팔레트 열기(`CMD/Ctrl + Shift + P`)
    2. "Preferences: Open Settings (UI)" 검색
    3. `workbench.activityBar.orientation` 검색
    4. 값을 `vertical`로 설정
    5. 변경 사항 적용을 위해 Cursor를 재시작
  </Step>
  <Step title="AI 패널 열기">
    Cursor 큐브 아이콘 버튼(AI Pane)을 클릭해 Cursor 에이전트(오른쪽 뷰 패널)를 여세요
  </Step>
  <Step title="AI 패널 사이드바에 Cline 드래그">
    Cline 아이콘을 AI 패널 사이드바로 직접 드래그하세요.
  </Step>
</Steps>
