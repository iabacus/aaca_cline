파일을 Cline으로 드래그 앤 드롭하면 이미지, 코드, 기타 파일을 대화에 빠르게 추가할 수 있습니다.

<Frame>
	<img
		src="https://storage.googleapis.com/cline_public_images/docs/assets/dran-n-drop.gif"
		alt="Cline 채팅에 파일 드래그 앤 드롭"
	/>
</Frame>

<Note>VS Code 특성상 Cline 채팅 입력에 파일을 드래그 앤 드롭할 때는 드래그하는 동안 `Shift` 키를 누르고 있어야 합니다.</Note>

워크스페이스 파일을 Cline에 드래그 앤 드롭하면 자동으로 [파일 멘션](/features/at-mentions/file-mentions)이 생성됩니다. 경로를 직접 타이핑하지 않고도 대화에서 파일을 참조할 수 있습니다.

시스템의 파일 관리자에서 Cline으로 파일을 직접 드래그할 수 있습니다:

### 지원 파일 형식

Cline은 워크스페이스 파일뿐 아니라, 시스템 파일에서 가져오는 외부 이미지, PDF, CSV, Excel, 기타 텍스트 파일 드래그를 지원합니다.
