description: "실험적 VS Code Language Model API를 통해 GitHub Copilot 및 호환 확장에서 모델에 접근하는 방법을 알아보세요."
---

Cline은 [VS Code Language Model API](https://code.visualstudio.com/api/extension-guides/language-model)를 _실험적으로_ 지원합니다. 이 API는 VS Code 환경 내에서 확장이 언어 모델에 직접 접근할 수 있도록 합니다. 따라서 다음과 같은 모델을 활용할 수 있습니다:

-   **GitHub Copilot:** Copilot 구독과 확장이 설치되어 있는 경우.
-   **다른 VS Code 확장:** Language Model API를 구현한 모든 확장.

**중요:** 이 통합은 실험 단계로, 기대한 대로 동작하지 않을 수 있습니다. 기능은 다른 확장이 VS Code Language Model API를 올바르게 구현하는지에 의존합니다.

### 사전 준비

-   **VS Code:** Language Model API는 VS Code에서만 사용 가능하며 현재 Cursor에서는 지원되지 않습니다.
-   **언어 모델 프로바이더 확장:** 언어 모델을 제공하는 확장이 필요합니다. 예시:
    -   **GitHub Copilot:** Copilot 구독이 있다면 GitHub Copilot과 GitHub Copilot Chat 확장이 모델 프로바이더로 동작합니다.
    -   **대체 확장:** VS Code Marketplace에서 "Language Model API" 또는 "lm"을 언급하는 확장을 찾아보세요. 다른 실험적 옵션이 있을 수 있습니다.

### 설정 단계

1. **Copilot 계정 활성화 및 확장 설치 확인:** Copilot 또는 Copilot Chat 확장에 로그인되어 있으면 Cline에서 접근할 수 있습니다.
2. **Cline 설정 열기:** Cline 패널의 톱니바퀴 아이콘(⚙️)을 클릭합니다.
3. **프로바이더 선택:** "API Provider" 드롭다운에서 "VS Code LM API"를 선택합니다.
4. **모델 선택:** Copilot 확장이 설치되어 있고 Copilot 계정에 로그인되어 있다면, 잠시 후 "Language Model" 드롭다운이 사용 가능한 모델로 채워집니다. 명명 규칙은 `vendor/family`입니다. 예시:
    - `copilot - gpt-3.5-turbo`
    - `copilot - gpt-4o-mini`
    - `copilot - gpt-4`
    - `copilot - gpt-4-turbo`
    - `copilot - gpt-4o`
    - `copilot - claude-3.5-sonnet` **참고:** 이 모델은 동작하지 않습니다.
    - `copilot - gemini-2.0-flash`
    - `copilot - gpt-4.1`

VSCode LM API Provider에서 최적의 결과를 위해 OpenAI 모델(GPT 3, 4, 4.1, 4o 등)을 사용하는 것을 권장합니다.

### 현재 제한 사항

-   **실험적 API 상태:** VS Code Language Model API는 아직 활발히 개발 중입니다. 변경과 불안정이 있을 수 있습니다.
-   **확장 의존성:** 이 기능은 다른 확장이 모델을 제공해야만 사용할 수 있습니다. Cline이 접근 가능한 모델 목록을 직접 제어하지 않습니다.
-   **제한된 기능:** VS Code Language Model API는 다른 API 프로바이더에서 제공되는 모든 기능(예: 이미지 입력, 스트리밍 응답, 상세 사용량 지표)을 포함하지 않을 수 있습니다.
-   **직접 비용 관리 불가:** 사용자는 모델을 제공하는 확장의 요금 및 이용 약관을 따르게 됩니다. Cline은 관련 비용을 직접 모니터링하거나 통제할 수 없습니다.
-   **GitHub Copilot 요율 제한:** GitHub Copilot과 함께 VS Code LM API를 사용할 때 GitHub의 요율 제한이 적용될 수 있습니다. 이 제한은 GitHub가 관리하며 Cline과는 무관합니다.

### 문제 해결 팁

-   **모델이 보이지 않을 때:**
    -   VS Code가 설치되어 있는지 확인합니다.
    -   언어 모델 프로바이더 확장(예: GitHub Copilot, GitHub Copilot Chat)이 설치 및 활성화되어 있는지 확인합니다.
    -   Copilot을 사용하는 경우, 원하는 모델로 Copilot Chat 메시지를 이전에 보낸 적이 있는지 확인하세요.
-   **예상치 못한 동작:** 예상치 못한 동작이 발생하면, Language Model API 또는 프로바이더 확장의 문제일 가능성이 큽니다. 해당 확장 개발자에게 문제를 보고하는 것을 고려하세요.
