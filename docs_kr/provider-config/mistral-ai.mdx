Cline은 Mistral AI API를 통해 표준 Mistral 모델과 코드 특화 Codestral 모델에 접근할 수 있습니다.

**웹사이트:** [https://mistral.ai/](https://mistral.ai/)

### API 키 받기

1.  **가입/로그인:** [Mistral Platform](https://console.mistral.ai/)으로 이동해 계정을 만들거나 로그인합니다. 검증 절차가 필요할 수 있습니다.
2.  **API 키 생성:**
    -   [La Plateforme API 키](https://console.mistral.ai/api-keys/) 및/또는
    -   [Codestral API 키](https://console.mistral.ai/codestral)

### 지원 모델

Cline은 다음 Mistral 모델을 지원합니다:

-   pixtral-large-2411
-   ministral-3b-2410
-   ministral-8b-2410
-   mistral-small-latest
-   mistral-medium-latest
-   mistral-small-2501
-   pixtral-12b-2409
-   open-mistral-nemo-2407
-   open-codestral-mamba
-   codestral-2501
-   devstral-small-2505

**참고:** 모델 가용성과 사양은 변경될 수 있습니다.
최신 정보는 [Mistral AI 문서](https://docs.mistral.ai/api/)와 [Mistral 모델 개요](https://docs.mistral.ai/getting-started/models/models_overview/)를 참고하세요.

### Cline에서 설정하기

1.  **Cline 설정 열기:** Cline 패널의 설정 아이콘(⚙️)을 클릭합니다.
2.  **프로바이더 선택:** "API Provider" 드롭다운에서 "Mistral"을 선택합니다.
3.  **API 키 입력:** 표준 `mistral` 모델을 사용하는 경우 Mistral API 키를 "Mistral API Key" 필드에 붙여넣습니다. `codestral-latest`를 사용하려면 아래 "Codestral 사용" 섹션을 참고하세요.
4.  **모델 선택:** "Model" 드롭다운에서 원하는 모델을 선택합니다.

### Codestral 사용

[Codestral](https://docs.mistral.ai/capabilities/code_generation/)은 코드 생성과 상호작용에 특화된 모델입니다.
Codestral은 다른 엔드포인트를 사용할 수 있습니다(기본값: codestral.mistral.ai).
La Plateforme API 키로 Codestral을 사용할 경우 **Codestral Base Url**을 `https://api.mistral.ai`로 변경하세요.

Cline에서 Codestral을 사용하려면:

1.  **Cline 설정에서 API Provider로 "Mistral"을 선택합니다.**
2.  **"Model" 드롭다운에서 Codestral 모델**(예: `codestral-latest`)을 선택합니다.
3.  **Codestral API 키**(`codestral.mistral.ai`) 또는 **La Plateforme API 키**(`api.mistral.ai`)를 Cline의 해당 API 키 필드에 입력합니다.
