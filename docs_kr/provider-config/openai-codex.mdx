OpenAI Codex를 사용하면 OAuth로 OpenAI 계정을 Cline에 직접 연결할 수 있어 API 키를 관리할 필요가 없습니다. 브라우저에서 한 번 클릭해 로그인하면 OpenAI 플랜에서 사용 가능한 모든 모델에 자동으로 접근할 수 있습니다. 추가 설정이 필요하지 않습니다.

<Tip>
API 키를 사용하고 싶다면 [OpenAI (API Key)](/provider-config/openai) 프로바이더 설정을 참고하세요.
</Tip>

## 사전 준비

- [OpenAI 계정](https://platform.openai.com/signup)
- 활성화된 OpenAI 구독 또는 API 접근 플랜

<Note>
사용 가능한 모델은 OpenAI 계정의 구독 등급에 따라 달라집니다. 각 플랜에 포함된 내용은 [OpenAI 가격 페이지](https://openai.com/pricing)를 참고하세요.
</Note>

## OpenAI 계정 연결

<Steps>
  <Step title="Cline 설정 열기">
    Cline 확장 오른쪽 상단의 설정 톱니바퀴 아이콘을 클릭합니다.
  </Step>
  <Step title="OpenAI Codex 프로바이더 선택">
    "API Provider" 드롭다운에서 **"OpenAI Codex"**를 선택합니다.
  </Step>
  <Step title="OpenAI로 로그인">
    **"Sign in with OpenAI"** 버튼을 클릭합니다. 인증을 위한 브라우저 창이 열립니다.
  </Step>
  <Step title="Cline 승인">
    열리는 브라우저 창에서:
    1. OpenAI 계정으로 로그인(이미 로그인되어 있다면 생략)
    2. Cline이 요청하는 권한 확인
    3. **"Authorize"**를 클릭해 접근 권한 부여
  </Step>
  <Step title="모델 선택">
    승인 후 Cline으로 돌아옵니다. **"Model"** 드롭다운에서 원하는 모델을 선택합니다.
  </Step>
</Steps>

<Note>
드롭다운에 모델이 보이지 않으면 OpenAI 구독 등급에서 해당 모델을 사용할 수 없을 수 있습니다.
</Note>

사용 가능한 최신 모델과 기능 목록은 공식 [OpenAI 모델 문서](https://platform.openai.com/docs/models)를 참고하세요.

## 연결 관리

### 계정 연결 해제

Cline에서 OpenAI 계정을 연결 해제하려면:

1. Cline 설정 열기
2. OpenAI Codex 프로바이더가 선택된 상태에서 **"Sign out"** 클릭
3. 연결 해제 확인

OAuth 연결이 제거됩니다. 언제든 다시 로그인해 재연결할 수 있습니다.

### 토큰 갱신

OAuth 토큰은 Cline이 자동으로 갱신합니다. 인증 오류가 발생하면:

1. 계정을 연결 해제한 뒤 다시 연결
2. OpenAI 계정 상태 확인
3. 구독이 활성 상태인지 확인

## 문제 해결

### "Authentication failed" 오류

- 올바른 OpenAI 계정으로 로그인했는지 확인
- OpenAI 구독이 활성 상태인지 확인
- 브라우저 캐시를 지우고 다시 로그인

### 모델이 표시되지 않음

- 사용 가능한 모델은 OpenAI 구독 등급에 따라 달라집니다
- 일부 모델은 특정 플랜 업그레이드가 필요할 수 있습니다
- 플랜별 모델 가용성은 [OpenAI 가격 페이지](https://openai.com/pricing)에서 확인하세요

### OAuth 창이 열리지 않음

- 브라우저의 팝업 차단을 확인
- 다른 브라우저로 시도
- 인터넷 연결 상태 확인

### 연결이 계속 만료됨

- 드물게 OpenAI 계정 세션이 만료되면 발생할 수 있습니다
- 연결을 해제하고 다시 연결해 인증을 새로고침하세요

## 알아두면 좋은 점

Cline은 필요한 최소 권한만 요청합니다. OpenAI 자격 증명은 Cline에 저장되지 않습니다.

OpenAI Codex를 사용할 때 Cline은 OpenAI/ChatGPT 구독을 통해 모델에 접근합니다. 이 프로바이더에서는 토큰 단위 API 과금이나 OpenAI API 크레딧 소모가 없으며, 사용량과 제한은 Cline이 아니라 ChatGPT(또는 OpenAI 계정) 구독 플랜에 따라 결정됩니다.
