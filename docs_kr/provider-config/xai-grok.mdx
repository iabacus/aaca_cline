xAI는 대화 능력과 큰 컨텍스트 창으로 알려진 대형 언어 모델 Grok의 개발사입니다. Grok 모델은 유용하고 정보성이 높으며 맥락에 맞는 응답을 제공하도록 설계되었습니다.

**웹사이트:** [https://x.ai/](https://x.ai/)

### API 키 받기

1.  **가입/로그인:** [xAI Console](https://console.x.ai/)에 접속해 계정을 만들거나 로그인합니다.
2.  **API Keys로 이동:** 대시보드의 API keys 섹션으로 이동합니다.
3.  **키 생성:** 새 API 키를 만들고 설명적인 이름(예: "Cline")을 지정합니다.
4.  **키 복사:** **중요:** API 키를 _즉시_ 복사하세요. 다시 확인할 수 없습니다. 안전하게 보관하세요.

### 지원 모델

Cline은 다음 xAI Grok 모델을 지원합니다:

#### Grok-3 모델

-   `grok-3-beta` (기본) - 131K 컨텍스트 창의 Grok-3 베타 모델
-   `grok-3-fast-beta` - 131K 컨텍스트 창의 Grok-3 fast 베타 모델
-   `grok-3-mini-beta` - 131K 컨텍스트 창의 Grok-3 mini 베타 모델
-   `grok-3-mini-fast-beta` - 131K 컨텍스트 창의 Grok-3 mini fast 베타 모델

#### Grok-2 모델

-   `grok-2-latest` - 131K 컨텍스트 창의 Grok-2 최신 모델
-   `grok-2` - 131K 컨텍스트 창의 Grok-2 모델
-   `grok-2-1212` - 131K 컨텍스트 창의 Grok-2 모델(버전 1212)

#### Grok Vision 모델

-   `grok-2-vision-latest` - 이미지 지원과 32K 컨텍스트 창을 가진 Grok-2 Vision 최신 모델
-   `grok-2-vision` - 이미지 지원과 32K 컨텍스트 창을 가진 Grok-2 Vision 모델
-   `grok-2-vision-1212` - 이미지 지원과 32K 컨텍스트 창을 가진 Grok-2 Vision 모델(버전 1212)
-   `grok-vision-beta` - 이미지 지원과 8K 컨텍스트 창을 가진 Grok Vision 베타 모델

#### 레거시 모델

-   `grok-beta` - 131K 컨텍스트 창의 Grok 베타 모델(레거시)

### Cline에서 설정하기

1.  **Cline 설정 열기:** Cline 패널의 설정 아이콘(⚙️)을 클릭합니다.
2.  **프로바이더 선택:** "API Provider" 드롭다운에서 "xAI"를 선택합니다.
3.  **API 키 입력:** xAI API 키를 "xAI API Key" 필드에 붙여넣습니다.
4.  **모델 선택:** "Model" 드롭다운에서 원하는 Grok 모델을 선택합니다.

### 추론 기능

Grok 3 Mini 모델은 "응답 전에 생각하기" 기능을 제공하며, 복잡한 문제 해결에 특히 유용합니다.

#### 추론 지원 모델

추론은 다음 모델에서만 지원됩니다:

-   `grok-3-mini-beta`
-   `grok-3-mini-fast-beta`

`grok-3-beta`와 `grok-3-fast-beta`는 추론을 지원하지 않습니다.

#### 추론 강도 조절

추론 지원 모델을 사용할 때 `reasoning_effort` 파라미터로 추론 강도를 조절할 수 있습니다:

-   `low`: 최소한의 사고 시간으로 빠르게 응답(적은 토큰 사용)
-   `high`: 최대한의 사고 시간으로 복잡한 문제를 해결(더 많은 토큰 사용)

간단한 질문에는 `low`, 복잡한 문제에는 `high`를 사용하세요.

#### 핵심 기능

-   **단계별 문제 해결**: 답변 전에 문제를 체계적으로 사고
-   **수학/정량 강점**: 수치 계산과 논리 퍼즐에 강함
-   **추론 트레이스 접근**: 모델의 사고 과정이 응답 완료 객체의 `reasoning_content` 필드를 통해 제공됨

### 팁과 참고사항

-   **컨텍스트 창:** 대부분의 Grok 모델은 최대 131K 토큰의 큰 컨텍스트 창을 제공하여 많은 코드와 컨텍스트를 포함할 수 있습니다.
-   **비전 기능:** 이미지 처리/분석이 필요하면 비전 지원 모델(`grok-2-vision-latest`, `grok-2-vision` 등)을 선택하세요.
-   **가격:** 모델별 가격이 다르며, 입력은 100만 토큰당 $0.3~$5.0, 출력은 100만 토큰당 $0.5~$25.0 수준입니다. 최신 가격은 xAI 문서를 참고하세요.
-   **성능 트레이드오프:** "Fast" 변형은 응답 속도가 빠르지만 비용이 높을 수 있고, "Mini" 변형은 더 경제적이지만 기능이 제한될 수 있습니다.
