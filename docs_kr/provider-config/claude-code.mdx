Claude Code 프로바이더를 사용하면 기존 Claude 구독을 Cline에서 활용할 수 있습니다. Claude Max 또는 Pro 구독이 있다면, 추가 API 비용 없이 Cline에서 Claude를 사용할 수 있습니다.

## 설정

먼저 시스템에 Claude Code를 설치하고 인증해야 합니다:

1. **Claude Code 설치**: Anthropic의 [공식 설정 가이드](https://docs.anthropic.com/en/docs/claude-code/setup)를 따라 Claude CLI를 설치하고 인증합니다.

2. **Cline에서 구성**:
    - Cline 설정 열기(⚙️ 아이콘)
    - **API Provider** 드롭다운에서 **Claude Code** 선택
    - Claude CLI 실행 파일 경로 설정(PATH에 있다면 보통 `claude`로 충분)

### Claude Code 경로 찾기

-   **macOS / Linux / WSL / Git Bash**: `which claude`
-   **Windows Command Prompt**: `where claude`

## 지원 모델

Claude Code 프로바이더는 다음 모델을 지원합니다:

-   `claude-sonnet-4-20250514` (추천)
-   `claude-opus-4-1-20250805`
-   `claude-opus-4-20250514`
-   `claude-3-7-sonnet-20250219`
-   `claude-3-5-sonnet-20241022`
-   `claude-3-5-haiku-20241022`

## 동작 방식

Cline에서 Claude Code를 사용하면 내부적으로 다음과 같이 동작합니다:

Cline은 Claude Code CLI를 래핑하여 요청을 처리합니다. 메시지를 보낼 때마다 Cline이 새 `claude` 프로세스를 시작하고, 대화를 전달한 뒤 응답을 스트리밍으로 반환합니다. AI 추론은 Claude Code가 수행하지만, 파일 편집, 터미널 명령 등 실제 도구 실행은 Cline이 담당합니다.

차이점으로는, 다른 프로바이더처럼 문자 단위로 실시간 스트리밍되지 않습니다. Claude Code는 전체 요청을 처리한 후 완성된 응답을 반환합니다.

## 제한 사항

Claude Code 사용 시 다음 사항을 유의하세요:

-   메시지에 포함된 이미지는 텍스트 플레이스홀더로 변환됩니다(Claude Code CLI는 이미지 업로드를 지원하지 않음)
-   프롬프트 캐싱을 지원하지 않음
-   응답이 다른 프로바이더처럼 실시간 스트리밍되지 않음

## 문제 해결

문제가 발생하면 다음을 확인하세요:

**인증 문제**: 구독 계정으로 Claude Code에 로그인되어 있는지 확인하세요. `claude auth status`를 실행해 확인할 수 있습니다.

**경로 문제**: Cline 설정에 있는 Claude CLI 경로가 올바른지 다시 확인하세요. 터미널에서 `claude --version`을 실행해 동작 여부를 확인합니다.

**그래도 문제가 있나요?** 이 통합 기능은 계속 개선 중입니다. [GitHub](https://github.com/cline/cline/issues)에 이슈를 등록하거나 [Discord](https://discord.gg/cline)에서 도움을 요청하세요.

## 구독과 함께 사용하기

Claude Max 구독이 있는 경우, Cline에서의 사용량은 추가 API 비용이 없으므로 결제 인터페이스에서 $0.00으로 표시됩니다. 사용량은 구독 제한에 포함되지만 토큰 단위 과금은 발생하지 않습니다.

구독으로 Claude Code를 사용하는 방법은 Anthropic 문서를 참고하세요:

-   [Claude Code Setup Guide](https://docs.anthropic.com/en/docs/claude-code/setup)
-   [Using Claude Code with Pro/Max Plans](https://support.anthropic.com/en/articles/11145838-using-claude-code-with-your-pro-or-max-plan)
