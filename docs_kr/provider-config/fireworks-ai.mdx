Cline은 Fireworks AI 플랫폼을 통해 다양한 최신 오픈소스 언어 모델에 빠르고 비용 효율적으로 접근할 수 있습니다. 속도와 안정성을 위해 설계된 Fireworks AI는 OpenAI 호환 API와 최대 256,000 토큰의 컨텍스트 창을 제공하는 서버리스 배포 옵션을 지원합니다.

**웹사이트:** [https://fireworks.ai/](https://fireworks.ai/)

### API 키 받기

1. **가입/로그인:** [Fireworks AI](https://fireworks.ai/)에 접속해 계정을 만들거나 로그인합니다.
2. **API Keys로 이동:** 로그인 후 계정 설정의 [API Keys 페이지](https://app.fireworks.ai/settings/users/api-keys)로 이동합니다.
3. **키 생성:** "Create API key"를 클릭하고 키에 설명적인 이름(예: "Cline")을 지정합니다.
4. **키 복사:** **중요:** API 키를 _즉시_ 복사하세요. 다시 확인할 수 없습니다. 안전하게 보관하세요.

### 지원 모델

Cline은 다음 Fireworks AI 모델을 지원합니다:

-   `accounts/fireworks/models/kimi-k2-instruct` (기본)
-   `accounts/fireworks/models/qwen3-235b-a22b-instruct-2507`
-   `accounts/fireworks/models/qwen3-coder-480b-a35b-instruct`
-   `accounts/fireworks/models/deepseek-r1-0528`
-   `accounts/fireworks/models/deepseek-v3`

**모델 상세:**

| Model | Context Window | Best For | Pricing (per 1M tokens) |
|-------|----------------|----------|-------------------------|
| Kimi K2 | 128K | 일반 작업, 에이전트 기능 | 입력 \$0.60, 출력 \$2.50 |
| Qwen3 235B | 256K | 비용 효율적 범용 사용 | 입력 \$0.22, 출력 \$0.88 |
| Qwen3 Coder | 256K | 코드 생성 및 디버깅 | 입력 \$0.45, 출력 \$1.80 |
| DeepSeek R1 | 160K | 복잡한 추론, 함수 호출 | 입력 \$3.00, 출력 \$8.00 |
| DeepSeek V3 | 128K | 강력한 범용 성능 | 입력 \$0.90, 출력 \$0.90 |

### Cline에서 설정하기

1. **Cline 설정 열기:** Cline 패널의 설정 아이콘(⚙️)을 클릭합니다.
2. **프로바이더 선택:** "API Provider" 드롭다운에서 "Fireworks AI"를 선택합니다.
3. **API 키 입력:** Fireworks AI API 키를 "Fireworks AI API Key" 필드에 붙여넣습니다.
4. **모델 선택:** "Model" 드롭다운에서 원하는 모델을 선택합니다. 기본 모델은 Kimi K2입니다.

### 팁과 참고사항

-   **비용 효율성:** Fireworks AI는 상용 모델 대비 훨씬 낮은 가격으로 경쟁력 있는 성능을 제공합니다.
-   **대규모 컨텍스트 창:** 대부분 모델이 128K~256K 토큰을 지원하여 대형 문서와 긴 대화를 처리하기에 적합합니다.
-   **OpenAI 호환성:** 스트리밍 지원과 사용량 추적을 포함한 OpenAI 호환 API 형식을 사용합니다.
-   **요율 제한:** Fireworks AI는 사용량 기반 요율 제한이 있습니다. 대시보드에서 사용량을 모니터링하고 필요 시 요금제를 업그레이드하세요.
-   **API 키:** 보안을 위해 로컬에 저장됩니다.
-   **가격:** 최신 가격은 [Fireworks AI pricing 페이지](https://fireworks.ai/pricing)를 참고하세요. 표기된 가격은 100만 토큰 기준입니다.
