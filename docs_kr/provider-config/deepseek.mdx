Cline은 DeepSeek API를 통해 `deepseek-chat`, `deepseek-reasoner` 등 모델에 접근할 수 있습니다.

**웹사이트:** [https://platform.deepseek.com/](https://platform.deepseek.com/)

### API 키 받기

1.  **가입/로그인:** [DeepSeek Platform](https://platform.deepseek.com/)에 접속해 계정을 만들거나 로그인합니다.
2.  **API Keys로 이동:** 플랫폼의 [API keys](https://platform.deepseek.com/api_keys) 섹션에서 API 키를 찾습니다.
3.  **키 생성:** "Create new API key"를 클릭하고 키에 설명적인 이름(예: "Cline")을 지정합니다.
4.  **키 복사:** **중요:** API 키를 _즉시_ 복사하세요. 다시 확인할 수 없습니다. 안전하게 보관하세요.

### 지원 모델

Cline은 다음 DeepSeek 모델을 지원합니다:

-   `deepseek-v3-0324` (코딩 작업 추천)
-   `deepseek-r1` (추론 작업 추천)

### Cline에서 설정하기

1.  **Cline 설정 열기:** Cline 패널의 ⚙️ 아이콘을 클릭합니다.
2.  **프로바이더 선택:** "API Provider" 드롭다운에서 "DeepSeek"를 선택합니다.
3.  **API 키 입력:** DeepSeek API 키를 "DeepSeek API Key" 필드에 붙여넣습니다.
4.  **모델 선택:** "Model" 드롭다운에서 원하는 모델을 선택합니다.

### 팁과 참고사항

-   **가격:** 모델 비용은 [DeepSeek Pricing](https://api-docs.deepseek.com/quick_start/pricing/) 페이지를 참고하세요.
